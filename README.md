# ObraConfiável (MVP Web)

Plataforma web para encontrar e contratar profissionais de obra e manutenção (pedreiros, serventes, eletricistas e outros) com foco em confiança e segurança.

## Funcionalidades do MVP

- Busca por tipo de serviço, cidade e região.
- Lista de profissionais com disponibilidade, avaliação e selo de verificação.
- Fluxo guiado de contratação segura:
  - contrato digital;
  - assinatura GOV para cliente e prestador;
  - pagamento protegido na plataforma (liberação após conclusão).
- Pré-cadastro de prestador com dados essenciais:
  - nome, telefone, e-mail, CPF;
  - documento de identidade;
  - local de atendimento.

## Próximos passos recomendados

- Autenticação com validação de telefone/e-mail.
- Onboarding completo com upload de documentos (CPF, identidade e comprovante de residência).
- Integração real com assinatura GOV.
- Motor de busca por geolocalização e raio de atendimento.
- Escrow/payment gateway com trilha de auditoria.
